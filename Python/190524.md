## 基礎語法

- Python2編碼為ASCII、Python3為utf8
	> 保險起見在開頭加上編碼宣告
	
	```
	#coding = utf-8
	```
	
- Windows換行為\r\n，Unix為\n
  
  > 在Windows編寫跟在Unix(樹梅派)上編寫會有轉換問題
  
- Print

  > 顯示輸出資料至螢幕

  ```python
  print("Hello, World")
  // Hello, World
  ```

- 型別轉換

  > x = 型別(輸入值)

  ```python
  x = int(input('第一個數字'))
  y = int(input('第二個數字'))
  print(x + y)
  # input 10 20
  # output 30
  
  """
  多行
  註解
  """
  ```
